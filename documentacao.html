Visão Geral

Este portfólio foi desenvolvido com foco em simplicidade, acessibilidade e responsividade, utilizando apenas HTML e CSS puros, sem frameworks.

Tecnologias Utilizadas

  • HTML5: Marcação semântica e estrutura da página.
  • CSS3: Layouts responsivos com grid e flexbox, variáveis CSS para paleta de cores.
  • JavaScript: Interatividade simples, como o botão de copiar cores na página da paleta.
  • Acessibilidade: Uso de tabindex, aria-current e foco visível para navegação facilitada.

Estrutura do Projeto

O projeto é organizado em páginas HTML separadas, cada uma focada em uma seção do portfólio, como:

  • Início
  • Sobre Mim
  • Portfólio (com galerias e descrições de projetos)
  • Contato
  • Paleta de Cores
  • Mapa do Site
  • Documentação (esta página)

Design e Estilo

O design é baseado em uma paleta de cores consistente, utilizando variáveis CSS para fácil manutenção. O layout é responsivo para garantir boa experiência em dispositivos móveis e desktops.

Melhorias Futuras
